# Aman Ali Pogaku Portfolio Website

Static portfolio website, tailored for DevOps and cloud engineering job search in the Canadian market.


## Features

- Responsive one-page portfolio.
- Sticky header navigation.
- Auto-hide floating `Top` button that appears while scrolling.
- Toronto skyline artwork built as inline SVG.
- Dark-mode skyline treatment with moon, building lights, and CN Tower-inspired night lighting.
- Contact links for email, LinkedIn, GitHub, and Blog.

## File Structure

```text
.
├── index.html
├── styles.css
└── .github/
    └── workflows/
        └── static.yml
```

## View Locally

Open `index.html` directly in a browser after downloading the repo.

## Maintenance Notes

- Edit main content in `index.html`.
- Edit layout, colors, responsive rules, and dark mode in `styles.css`.
- The skyline artwork is inline SVG inside the hero section of `index.html`.
- The theme toggle and auto-hide scroll button script are at the bottom of `index.html`.
- Keep new content factual and resume-backed.
